# go.yondar.me

Go and check out the Problem tree here: https://nostrocket.org/problems/rocket/yondar

### Installation

1. **Clone the Repository:**

    ```bash
    git clone https://github.com/innovatario/yondar-mono.git

    cd yondar-mono
    
    npm install

    npm run dev
    ```
